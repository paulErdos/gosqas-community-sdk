## How to Get Started 

1. Fork this repo,

<img width="2327" height="502" alt="Screenshot 2025-11-03 at 04 19 38" src="https://github.com/user-attachments/assets/85a54abb-2998-4988-9c66-dd584097bff8" />

2. Clone, cd, ./install.sh
3. Run ./gdt.ts.
4. To get an idea of how to extend this to suit bespoke needs, saunter
through [gdt-utils.ts](https://github.com/paulErdos/gosqas-community-sdk/blob/main/gdt-utils.ts).


## How to Keep Going

Refer to [the GDT Integration Tests](https://github.com/gosqasorg/asset-provenance-tracking/tree/main/packages/backend/test/IntegrationTests/Live). These are their source of truth, and feature a lot of reusable code.
